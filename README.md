# PayNexa — Responsive E-Wallet Web App

Prototype web application e-wallet menggunakan:

- HTML5
- CSS3
- Bootstrap 5
- Bootstrap Icons
- JavaScript Vanilla

## Cara Menjalankan

1. Extract file ZIP.
2. Buka folder `paynexa-ewallet`.
3. Klik dua kali file `index.html`.
4. Login menggunakan input bebas, contoh:
   - Email: `rexy@email.com`
   - Password: `123456`

## Struktur File

```text
paynexa-ewallet/
├── index.html
├── style.css
├── script.js
└── README.md
```

## Fitur Prototype

- Splash screen
- Login dan register
- Dashboard user
- Card saldo
- Top up saldo
- Transfer saldo dengan validasi saldo
- QRIS scan simulation
- Riwayat transaksi dengan search dan filter
- Detail transaksi
- Notifikasi
- Profil pengguna
- FAQ dan live chat CS simulasi
- Admin dashboard
- Kelola pengguna admin
- Kelola merchant
- Laporan transaksi admin
- Responsive mobile-first
- Sidebar desktop dan bottom navigation mobile

Catatan: ini adalah prototype frontend tanpa backend/database.
